# Daily AI Productivity Logger

A simple Python CLI tool to track daily AI learning goals and progress.

## Features

* Save daily goals with a timestamp
* Read all past logs
* Lightweight and easy to use

## Tech Stack

* Python (standard library only)

## Usage

Run the script:

```
python main.py
```

Choose:

* `1` → View logs
* `2` → Add new log

## Purpose

Building consistency in coding and AI development through daily tracking.

## Future Improvements

* AI-based log summarization
* Weekly analytics
* Integration with LLM APIs
